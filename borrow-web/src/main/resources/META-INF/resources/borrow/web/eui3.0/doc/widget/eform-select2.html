下拉框

下拉框分为3大类型：eform下拉框、ecombobox下拉框、epicker拾取下拉框。

 该页面仅仅为下拉框的使用方法，请点击表单综合应用查看。

一、eform下拉框的共性为，都有两种实现方式，具体可展开代码详解；
模块：eui/modules/eform。

LISTBOX ECOMBOBOX流式列表下拉框

对象名：EListBoxCombobox；流式列表下拉框，下拉框内部采用elistbox实现。

展开代码 复制代码 
//流式列表下拉框
require(["eui/modules/ecombobox"], function(ecombobox){
    var datas4 = [{src:"../../../eui/doc/images/icon_tjt04.png"},{src:"../../../eui/doc/images/icon_tjt03.png"}];
    comboboxobj = new ecombobox.EListBoxCombobox({
        wnd: window,
        parentElement: document.getElementById("elistboxcombobox"),
        width: 240,
        height: "100%",
        floatBasecss: "eui-panel-listbox",
    });
    var listbox = comboboxobj.getListBox();
    EUI.addClassName(listbox.getContent(), "eui-listbox-vertical");
    listbox.addGroupElement(false, "");
    datas4.forEach(function(icondata, index){
        var item = listbox.addItem('<i class="eui-icon eui-icon-img" style="background-position: left;background-image:url('+icondata.src+');"></i>');
        item._customvalue = icondata.src;
    });
    comboboxobj.setItemIndex(0);
    EUI.addDispose(comboboxobj);
});

三、epicker拾取下拉框共性为：大部分具有type属性，可通过type属性改变下拉框大、中、小的型号；

大：为默认下拉框，无值；使用场景：在位置比较宽松时使用该拾取框；
中：；使用场景：位置比较紧凑时提供了此风格使用；
小：；使用场景：在位置比较狭小，如工具栏上使用；

模块：eui/modules/epicker。
